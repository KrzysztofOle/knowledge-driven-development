# KGAID experience

Ten obszar przechowuje robocze, audytowalne zapisy doświadczeń z projektów
stosujących KGAID. Nie jest częścią przygotowanego baseline'u
`KGAID-0.1.0` i nie zmienia normy.

- [Model Experience Record](experience-record-model.md) — ogólny projekt
  struktury rekordu doświadczenia.
- [Rejestr projektów](reference-projects/README.md) — projekty, niezmienne
  rewizje, status referencyjny i dostępne rekordy.
- [Macierz obserwacji i reakcji](observation-response-matrix.md) — kontrola
  kompletności reakcji na obserwacje.
- [Propozycja lifecycle feedbacku](feedback-lifecycle-proposal.md) — roboczy
  model obsługi informacji zwrotnej.
- [Proces ewolucji metodyki](../50-governance/evolution-workflow.md) — synteza
  drogi od doświadczenia projektu przez review i decyzję do nowego baseline'u.
- [EXP-3KSEF-001](reference-projects/3ksef/EXP-3KSEF-001.md) — pierwszy rekord,
  oparty na pilocie `3ksef`.

Dokument źródłowy pozostaje evidence. Experience Record streszcza go i wiąże
z propozycjami zmian, lecz go nie zastępuje ani nie przekształca w dokument
normatywny.
