## Kontrolowany słownik statusu

- `pilot` — pojedyncze użycie analizowane bez autorytatywnego statusu;
- `candidate` — kandydat oceniany według ustalonych kryteriów;
- `reference` — status nadany formalną decyzją Human Authority;
- `archived` — rekord historyczny, niewykorzystywany do bieżącej walidacji.

Rejestr nie nadaje sam statusu. Pole odzwierciedla wyłącznie zapisaną decyzję
albo bezpieczny stan początkowy `pilot`.
