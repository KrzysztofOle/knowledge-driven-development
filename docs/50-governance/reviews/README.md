# KGAID reviews

Ten katalog zawiera przeglądy i assessment dotyczące metodyki. Review ocenia
wskazany przedmiot i może przedstawiać rekomendację, ale nie podejmuje decyzji
Human Authority, nie zmienia statusu dokumentów i nie zatwierdza baseline'u.

- [Architecture Impact Review dla KGAID-CP-001](architecture-impact-review-cp-001.md)
  — eksperymentalna ocena wpływu rozdzielenia statusu dokumentu, wiedzy i
  technical approval.
- [Baseline Readiness Assessment](baseline-readiness-assessment.md) — ocena
  kompletności, spójności, traceability, governance, praktycznej walidacji,
  otwartych kwestii i ryzyk aktualnego stanu KGAID.
